# 💻 Justke CLI

> Não é só um portfólio. É uma experiência.

---

## 🧠 História

Tudo começou com uma brincadeira.

Eu e uma colega temos o costume de criar desafios um pro outro — sempre com o objetivo de aprender algo novo e sair da zona de conforto. E o desafio da vez era simples:

Criar um portfólio.

Mas não qualquer portfólio.

A regra era clara: tinha que ser algo diferente, algo que não se vê todo dia.

Foi então que ela lançou o dela… feito em **Assembly**.

Naquele momento eu pensei:
“Ok… isso aqui já está em outro nível.”

Assembly é baixo nível, direto com o hardware, algo que poucos se arriscam a fazer. E foi aí que veio a resposta — meio na brincadeira, mas com ambição por trás:

> “Então eu vou fazer um sistema operacional.”

A ideia parecia absurda. E talvez fosse mesmo.

Mas antes de chegar lá, eu pensei:
“Vou começar brincando um pouco com algo menor.”

E assim nasceu o **Justke CLI**.

Uma interface de terminal interativa, com animações, comandos e uma experiência diferente de tudo que um portfólio tradicional entrega.

Foi construído usando Node.js, aplicando conhecimentos que eu já tinha e explorando novas ideias no processo. Mais do que aprender uma nova tecnologia, o foco foi criar algo com personalidade.

Mas isso aqui ainda não é o final.

O CLI é só o começo.

---

## 😈 O que vem depois

O próximo passo já está definido:

**JustkeOS**

Um portfólio em formato de sistema operacional.

Bootável. Em ISO. Fora do padrão.

A ideia não é competir com Assembly.

É contornar.

Se não dá pra superar no nível da linguagem…

Então a solução é mudar completamente o jogo.

---

## 🚀 Sobre o projeto

O **Justke CLI** é uma ferramenta de linha de comando que apresenta meu portfólio de forma interativa.

Você não navega por páginas.

Você executa comandos.

---

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/RichardtJustke/justke-cli.git
````

Acesse a pasta:

```bash
cd justke-cli
```

Instale as dependências:

```bash
npm install
```

Link global:

```bash
npm link
```

---

## 🧠 Como usar

```bash
justke
```

Ou comandos diretos:

```bash
justke about
justke projects
justke skills
justke services
justke experience
justke ketsu
justke justkeOS
justke contact
justke help
```
